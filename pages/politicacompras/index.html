Como funcionam os preços para lojistas

Os produtos para lojistas possuem preços por quantidade.

Isso significa que, quanto maior a quantidade comprada dentro do mesmo Grupo de Quantidade, melhor pode ficar o valor unitário dos produtos daquele grupo.

Importante: o Grupo de Quantidade serve para somar as unidades compradas. O preço de cada produto continua sendo o preço informado na tabela daquele próprio produto.


O que é Grupo de Quantidade?

Cada produto pertence a um Grupo de Quantidade.

Exemplos de grupos:

  • Dados Acrílicos
  • Dados Runas
  • Dados Premium
  • Dados Exóticos
  • Bandeja de Rolagem
  • Estojo de Dados
  • Torres de Rolagem
  • Miniaturas

Produtos do mesmo grupo podem ser somados para atingir uma faixa melhor de quantidade.

Mas isso não significa que todos os produtos do grupo tenham o mesmo preço.


Como ler a tabela de cada produto

Em cada produto, você verá uma informação parecida com esta:

Grupo de Quantidade: Dados Acrílicos

  • 15 unidades = R$ 15,90 cada
  • 30 unidades = R$ 14,90 cada
  • 60 unidades = R$ 13,90 cada

Isso quer dizer que aquele produto usa o grupo Dados Acrílicos para contar a quantidade.

Se você comprar mais produtos do mesmo grupo, eles ajudam a alcançar uma faixa melhor.


Exemplo 1: produtos do mesmo grupo

Imagine uma compra com:

  • 15 Dados Translúcidos
  • 15 Dados Perolados

Os dois produtos fazem parte do grupo Dados Acrílicos.

Então, para calcular a faixa de quantidade, somamos:

15 + 15 = 30 unidades em Dados Acrílicos

Como a compra atingiu 30 unidades no grupo Dados Acrílicos, os produtos entram na faixa de 30 unidades.

Neste exemplo:

  • Dados Translúcidos entram no valor da faixa de 30 unidades informado no próprio produto.
  • Dados Perolados entram no valor da faixa de 30 unidades informado no próprio produto.

Se os dois produtos tiverem a mesma tabela, os dois terão o mesmo valor. Se algum produto tiver uma tabela diferente, ele seguirá a própria tabela.


Exemplo 2: produtos de grupos diferentes

Imagine uma compra com:

  • 15 Dados Perolados
  • 15 Runas de Dragão

Dados Perolados fazem parte do grupo Dados Acrílicos.

Runas de Dragão fazem parte do grupo Dados Runas.

Como são grupos diferentes, eles não são somados na mesma faixa.

O cálculo fica separado:

  • 15 unidades no grupo Dados Acrílicos
  • 15 unidades no grupo Dados Runas

Cada produto será calculado conforme a tabela do seu próprio grupo e do seu próprio produto.


Exemplo 3: produtos exóticos

Imagine uma compra com:

  • 3 Dados Metal Gear
  • 3 Dados Metal Wheel
  • 3 Dados Metal Spinner 6 in 1

Esses produtos fazem parte do grupo Dados Exóticos.

Então, para calcular a faixa de quantidade, somamos:

3 + 3 + 3 = 9 unidades em Dados Exóticos

Com 9 unidades no grupo Dados Exóticos, os produtos entram na faixa de 9 unidades.

Cada produto será calculado pelo valor da faixa de 9 unidades informado na sua própria tabela.


Resumo da regra de quantidade

Para calcular o preço, seguimos esta ordem:

  1. Verificamos o Grupo de Quantidade de cada produto.
  2. Somamos as unidades compradas dentro de cada grupo.
  3. Identificamos qual faixa de quantidade foi atingida em cada grupo.
  4. Aplicamos o valor correspondente na tabela de cada produto.

O grupo ajuda a definir a faixa de quantidade. O valor final vem da tabela de cada produto.


Pedido mínimo

O pedido mínimo para lojistas é de R$ 500,00.


Frete e retirada

A taxa fixa de frete é de R$ 100,00.

Pedidos a partir de R$ 3.000,00 têm frete grátis.

Retirada no local é grátis.


Formas de pagamento

A condição padrão da tabela é boleto 15/30 dias.

  • PIX: 2% de desconto
  • Boleto 15/30 dias: preço de tabela
  • Outras condições de pagamento: podem ter ajuste financeiro informado na cotação

Resumo geral

O cálculo do pedido funciona assim:

  1. Somamos as quantidades por Grupo de Quantidade.
  2. Aplicamos a faixa atingida em cada grupo.
  3. Cada produto mantém o preço da sua própria tabela.
  4. Somamos o valor dos produtos.
  5. Aplicamos a condição de pagamento escolhida.
  6. Aplicamos o frete, se houver.

Antes da confirmação do pedido, o lojista recebe a cotação completa com todos os valores.
